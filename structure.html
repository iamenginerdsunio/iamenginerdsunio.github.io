Important Directories

Directory Purpose
templates Holds Jinja2 HTML templates for the Flask UI, including the shared layout (base.html), the main index page (index.html), and section-specific templates grouped under books/, loans/, and members/.
templates/books Templates related to book management views (e.g., listing, creating, updating, deleting books).
templates/loans Templates for managing loan-related views such as borrowing and returning books.
templates/members Templates for managing library members (e.g., registering and editing member details).
static Container for static assets such as CSS, JavaScript, and images served by Flask; specific files not listed in the snapshot.
tests pytest-based test suite validating the Flask application behavior, with test_app.py exercising routes and core flows.
docs Documentation artifacts, currently including newfile.md as a placeholder or example doc.
