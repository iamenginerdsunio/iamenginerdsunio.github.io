Library Management System

Project Purpose

A simple library management system to manage books, members, and loans with CRUD views and borrowing/return workflows.

Project Type

Flask-based web application with HTML templates and CSS assets.

Primary Languages and Frameworks

  • Languages: Python, HTML, CSS
  • Frameworks: Flask, Flask-SQLAlchemy, Flask-Migrate

Architecture Summary

Single Flask application module (app.py) using SQLAlchemy models (models.py), application-wide extensions (extensions.py), Jinja2 templates organized by domain, and pytest-based tests.

Quick Setup

  1. Install dependencies from requirements.txt.
  2. Run the Flask application using app.py.

Getting Started Notes

Follow the setup steps to initialize the app. Templates for books, members, and loans are available under the templates directory for quick customization.
