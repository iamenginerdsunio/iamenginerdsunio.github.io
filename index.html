Library Management System

A robust Flask-based library management application.

Language: Python Framework: Flask License: MIT

Project Purpose

The Library Management System is a web application designed for managing library operations such as tracking books, users, and loans. Built using Flask, it employs SQLAlchemy for Object-Relational Mapping (ORM), enabling efficient database interactions.

The application follows the MVC (Model-View-Controller) architecture, separating concerns across routes, models, and templates. It provides a modular and scalable solution for libraries of all sizes.

Targeted at library administrators and staff, it aims to simplify daily operations while ensuring scalability and flexibility for future enhancements.

Key Features

  • Flask factory pattern for modular app initialization.
  • SQLAlchemy ORM for database management.
  • CRUD API endpoints for managing books and users.
  • Unit testing with pytest and Flask test client.
  • Clear separation of concerns in MVC architecture.

Tech Stack

  • Primary Language: Python
  • Frameworks: Flask
  • Database: SQLAlchemy ORM
  • Deployment: Not detected

Architecture Overview

The application uses an MVC architecture to organize models, views, and controllers. Models are defined in models.py using SQLAlchemy, representing database entities such as books and users. Views are rendered using HTML templates stored in the templates directory, while controllers in app.py handle routing and API logic.

Quick Start

Prerequisites

  • Python 3.7+
  • pip installed

Installation Steps

# Clone the repository
git clone https://github.com/iamenginerdsunio/LibManagement.git

# Navigate to the project directory
cd LibManagement

# Install dependencies
pip install -r requirements.txt

Basic Usage

# Run the application
python app.py

# Access the application at http://localhost:5000
