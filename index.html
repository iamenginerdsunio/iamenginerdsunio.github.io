Library Management System

Project Purpose

A minimal library management system to manage books, members, and loans with CRUD operations and borrowing workflows.

Project Type

Flask-based web application with HTML templates, CSS assets, and automated tests.

Primary Languages

  • Python
  • HTML
  • CSS

Frameworks

  • Flask
  • Flask-SQLAlchemy
  • Flask-Migrate
  • pytest
  • pytest-flask

Architecture Summary

Single Flask application module (app.py) with SQLAlchemy models (models.py), shared extensions (extensions.py), Jinja2 templates grouped by domain (books, members, loans), static CSS, and pytest tests.

Quick Setup

Dependencies are listed in requirements.txt. Install them using pip, set up environment variables, initialize database migrations, and run the Flask application.

Getting Started

Start by exploring app.py for routes and models.py for core entity definitions. Jinja2 templates in templates provide the UI structure, while tests in tests/test_app.py validate functionality.
