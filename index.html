Project Overview

Purpose

Provide a basic library management system to manage books, members, and loans (borrow/return) via a web interface.

Project Type

Web application (server-rendered) for library management.

Primary Languages

Python, HTML.

Frameworks

Flask, Flask-SQLAlchemy, Flask-WTF.

Architecture Summary

The application is a single-module Flask app (app.py) with:

  • SQLAlchemy models defined in models.py.
  • Jinja2 templates grouped by domain (books, members, loans).
  • Reusable extensions initialized in extensions.py.
  • Tests located in the tests/ package.

Quick Setup

Follow these steps to set up the project:

  1. Clone the repository: git clone https://github.com/iamenginerdsunio/LibManagement.git
  2. Navigate to the project directory: cd LibManagement
  3. Create a virtual environment: python -m venv venv
  4. Activate the virtual environment:
    • Windows: venv\Scripts\activate
    • macOS/Linux: source venv/bin/activate
  5. Install dependencies: pip install -r requirements.txt
  6. Run the application: python app.py

Getting Started

Once the application is running, visit http://127.0.0.1:5000 in your browser to access the library management system. Use the navigation menu to explore books, members, and loans.
