Modules

Dependencies

  • Flask
  • Flask-SQLAlchemy
  • SQLAlchemy
  • pytest
  • requests

Key Modules

app (app.py)
Creates and configures the Flask application, initializes extensions, defines routes for books, members, and loans, and wires templates and models together.
models (models.py)
Defines SQLAlchemy ORM models for core entities such as books, members, and loans, including fields and relationships used by the app.
extensions (extensions.py)
Declares Flask extension instances (e.g., SQLAlchemy db object) for import into app.py and models.py.
tests.test_app (tests/test_app.py)
Provides pytest-based tests that exercise the Flask routes and core library management workflows.

Data Models

  • Book model: Defined in models.py via SQLAlchemy, used in book listing/detail/form templates.
  • Member model: Defined in models.py, used in member listing/detail/form templates.
  • Loan model: Defined in models.py, used in loan listing and borrow templates.
