Modules

Key Modules

Module Path Responsibilities
app app.py Creates and configures the Flask application, sets up database and extensions, defines routes for books, members, and loans, and renders templates.
models models.py Defines SQLAlchemy ORM models for key entities (e.g., Book, Member, Loan) and encodes their relationships and persistence logic.
extensions extensions.py Initializes shared Flask extensions like the SQLAlchemy database instance for reuse in app.py and models.py.
tests.test_app tests/test_app.py Contains pytest tests to validate routes, CRUD operations, and response behaviors using Flask’s test client.
templates.base templates/base.html Defines the base layout with shared structure and Jinja2 blocks for content, used by other templates via inheritance.
templates.index templates/index.html Implements the home page UI, summarizing library data and linking to management pages for books, members, and loans.

Dependencies

  • Flask: Core web framework for routing, request handling, and response generation.
  • Flask-SQLAlchemy: ORM library for database operations and model definitions.
  • pytest: Testing framework used to validate application functionality.
  • Other dependencies: Implied in requirements.txt, but not explicitly enumerated in the data.
