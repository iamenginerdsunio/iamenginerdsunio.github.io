Modules

Dependencies

  • Flask
  • Flask-SQLAlchemy
  • Flask-Migrate
  • pytest

Key Modules

app (app.py)
Creates and configures the Flask application, initializes database/migration extensions, defines routes for index, book, member, and loan operations, and renders templates.
models (models.py)
Defines SQLAlchemy ORM models for library entities (Book, Member, Loan) and their relationships, including fields for book metadata, member details, and loan associations.
extensions (extensions.py)
Holds extension instances (e.g., SQLAlchemy db, migration handler) that are imported and initialized by app.py and used within models.py.
tests.test_app (tests/test_app.py)
Implements pytest-based tests for the Flask app, including client fixtures and assertions covering route responses and domain workflows (books, members, loans).
templates/base (templates/base.html)
Shared base layout with common HTML structure and Jinja2 blocks extended by other templates.

Responsibilities

  • app.py: Core application setup, routing, and template rendering.
  • models.py: ORM models for Book, Member, and Loan entities, with database relationships.
  • extensions.py: Configuration and initialization of Flask extensions.
  • tests/test_app.py: Ensures correct implementation of key functionalities and workflows.

Data Model Notes

  • Book: Represents library books with fields such as title, author, and availability-related attributes.
  • Member: Represents library members with identifying and contact fields.
  • Loan: Links books and members to track borrow transactions, including dates and status.
